# CurrencyRateProject
*Pet project*<br/>

It's an application, where you can view currency exchanging rates relative to Ruble.<br/>
The purpose of creating this application was learning how to work with API.

## Technology stack:
- Retrofit2
- Kotlin coroutines
- RecyclerView
- MVVM
- Clean Architecture
- Android ViewModel
- SimpleXmlConverter
